# My Fullstack Application - Server

This README provides information about the server-side of the full-stack application built with Node.js and Express.

## Project Structure

- **src/**: Contains the source code for the server.
  - **app.ts**: The main entry point for the server application. Sets up the Express server and middleware.
  - **routes.ts**: Defines the API routes for the application.

## Installation

1. Navigate to the server directory:
   ```
   cd server
   ```

2. Install the dependencies:
   ```
   npm install
   ```

## Running the Server

To start the server, run the following command:
```
npm start
```

The server will be running on `http://localhost:5000` (or the port specified in your configuration).
